# Streamlit Manajemen Toko

## Cara menjalankan lokal
```bash
pip install -r requirements_streamlit.txt
streamlit run streamlit_manajemen_toko.py
```

## Akun login
Password admin: `Store01`

## Fitur
- Login admin
- Tambah produk
- Daftar produk
- Cari produk
- Jual produk
- Update stok
- Simpan data JSON
- Close shift

## Catatan deployment
Simpan file berikut dalam satu folder:
- `streamlit_manajemen_toko.py`
- `requirements_streamlit.txt`

Lalu jalankan dari environment Python yang memiliki Streamlit.
